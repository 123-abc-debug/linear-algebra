# 向量空间
封闭 对线性组合封闭

1.定义：向量空间是满足特定规则的向量集合，核心规则是对线性运算（向量相加、向量数乘）封闭，即对空间内任意向量 v、w，运算结果 3v（数乘）、v+w（相加）仍在该空间中。
2.特点：重要特征：零向量必存在于所有向量空间中，因为任何向量乘 0 或加其反向向量都得到零向量，而线性运算封闭性要求零向量在空间内。
3.反例：
*R*2中仅第一象限的区域 D 不是向量空间。因对区域内向量进行数乘运算时，若乘数为负数，得到的向量会进入第三象限，脱离区域 D，不满足线性运算封闭性。

## 子空间
定义：子空间是向量空间中满足线性运算封闭性的部分，即子空间本身也是一个向量空间。
子空间 = 某些向量的所有线性组合构成的集合。
关键条件：子空间必须包含零向量，且对线性运算（相加、数乘）封闭，二者缺一不可（如不过原点的直线不包含零向量，不是子空间）。

举例：
其常见子空间有三类，一是过原点的平面（平面内向量运算封闭，含零向量），二是过原点的直线（直线上向量运算封闭，含零向量），三是零向量空间（仅含零向量，满足封闭性）。

# 如何从矩阵的角度来理解向量空间
矩阵的作用对象是向量空间，矩阵的输出也落在某个向量空间。
向量空间 = 矩阵可以作用的“舞台”

# 列空间是什么意思？这里的空间和学习的矩阵之间的联系是什么
列空间 = 矩阵 A 的列向量能“铺出来的整个空间”。
是它们的所有线性组合形成的子空间。
## 本质
矩阵本质上是“向量的集合”
列空间是“矩阵能把向量 Ax 推到的所有位置的集合”。



3. 列空间为什么重要？（你之后会用到）

你以后会学习：

AX = b 有解 ⇔ b ∈ 列空间 C(A)

也就是说：

左边 AX = b 就是在问：
“能不能用 A 的列向量拼出 b？”

b 在列空间里 → 有解

b 不在列空间里 → 无解

这就是列空间的核心意义。
