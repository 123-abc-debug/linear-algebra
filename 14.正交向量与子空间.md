# 核心概念
感觉这节课的核心是告诉我们 Ax=0的含义是将整个空间一分为二为两个相互正交的空间

并且老师从一个三维空间举例子，来想一下怎么把三维空间拆解的

接着又提到了正交补这个概念，但是我没注意到为什么会提到

零空间包含所有向量垂直于行空间的

给一个子空间 $S\subset \mathbb R^n$，
 它的 **正交补 $S^\perp$** 定义为：
$$
S^\perp = \{\, x\in\mathbb R^n : x\cdot s = 0 \text{ 对所有 } s\in S \,\}
$$
也就是说，**正交补中的向量与整个子空间所有向量都垂直**。

一个关键结论：

> **任意空间都可以分成两个互相垂直的子空间：它本身 + 它的正交补。**
