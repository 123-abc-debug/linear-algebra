# 标准正交基
四、标准正交基 这部分是引出下节的部分，内容较少，了解即可。
之前见过[ 1 0 0 ]，[ 0 0 1 ]，[ 0 1 0 ]这组基，它们显然是正交的，
但是它们还有更特殊的 性质，即它们都是单位向量，长度为 1。

所以这里我们引入一个新名词：**标准正 交向量组，**
其中的“标准”表示单位向量。 同样的标准正交向量组还有：[ 𝑐𝑜𝑠𝜃 𝑠𝑖𝑛𝜃 ]，[ −𝑠𝑖𝑛𝜃 𝑐𝑜𝑠𝜃 ]
